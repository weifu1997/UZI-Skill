## 🔴 P0 关键问题（必须立即修复）

### 1. API 密钥泄露风险
- **影响**: 密钥可能暴露在日志、错误消息中
- **修复时间**: 2天
- **工具**: `lib/security.py` (已创建)

### 2. mini_racer 崩溃
- **影响**: macOS Python 3.12+ 环境进程崩溃
- **修复时间**: 2天
- **方案**: 替换为纯HTTP实现

### 3. 双架构债务
- **影响**: 维护成本翻倍，开发速度减半
- **修复时间**: 2周
- **方案**: 统一到 pipeline 架构
